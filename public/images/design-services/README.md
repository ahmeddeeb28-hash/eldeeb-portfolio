# Design Services — Image Slots

Drop files in this folder and reference them from
`components/DesignServices.tsx` → the `DESIGN_SERVICES_IMAGES` object at the
top of the file.

| Slot key        | Service card                         | Suggested filename                |
| --------------- | ------------------------------------ | --------------------------------- |
| `strategy`      | 01 · Strategy — Product UX & Strategy | `01-strategy.jpg`                 |
| `craft`         | 02 · Craft — End-to-End Product Design | `02-craft.jpg`                   |
| `systems`       | 03 · Systems — Design Systems & Components | `03-systems.jpg`             |
| `growth`        | 04 · Growth — UX Optimization & Growth | `04-growth.jpg`                  |

Recommended: 16:10 landscape, 1200×750 px, JPG at quality 80, neutral tones
that sit comfortably on the #0A0A0A background. Black-and-white / low-sat
photography works best so the teal accent stays the only pop of colour.

To use local files, set the value to e.g. `/images/design-services/01-strategy.jpg`.
To use Unsplash / remote, paste the full URL.
